# Conventions

## Parentheses notation in task assignments

When defining aliases or navigation targets, parentheses around a letter indicate the **short (single-letter) alias**. The full word is the **long alias**.

### Format

```
(x)name     → short alias: x, long alias: name
prefix(x)   → short alias: x, long alias: prefixx...
```

### Examples

| Notation       | Short alias | Long alias   | Target                  |
|----------------|-------------|--------------|-------------------------|
| `(s)tack`      | `s`         | `stack`      | ~/jobs/solidpixels/stack |
| `(z)sh`        | `z`         | `zsh`        | ~/Documents/apps/shell  |
| `d(o)cuments`  | `o`         | `documents`  | ~/Documents             |
| `down(l)oads`  | `l`         | `downloads`  | ~/Downloads             |
